# Обзор источников для книги: журналы, газеты и статьи о демосцене ZX Spectrum

> Составлено из архива ZXArt (zxart.ee), Hype (hype.retroscene.org) и смежных ресурсов.
> Фокус: технические статьи о кодинге, дизайне демо, making-of, интервью со сценерами.

---

## 1. Ключевые журналы на ZXArt (zxart.ee/eng/software/pressa/)

### ★★★ Spectrum Expert #01–#02 (X-Trade, СПб, 1997–1998)
- **Уже есть в плане книги.** Два номера, оба на ZXArt.
- Авторский коллектив: Dark (Виталий Видмиров) — код + статьи, LD (Дмитрий Ломов) — код, Dangerous (Вячеслав Сарафов) — главред.
- Музыка: Frunze, Vega, Rush, STS, X-agon, Ironman, Panda.
- SE#01: алгоритмы умножения, деления, синуса, Брезенхем. SE#02: 3D графика, метод средних точек, HiSoft C.
- **Ценность:** Прямой источник от авторов Illusion. Фундамент Part I и Part III½ книги.
- URL: `zxart.ee/eng/software/series/spectrum-expert/`

### ★★★ ZX Format #01–#08 (XL Design, СПб, 1995–1998)
- 9 номеров (8 вышло + невыпущенный #9). Один из самых известных электронных журналов.
- Распространялся студией «Логрос».
- **Что искать:** статьи по ассемблеру (номер #02 содержит вводную статью по ассемблеру: флаги, арифметика, регистры), обзоры железа (Scorpion ZS-256-Turbo, ZX General Sound от X-Trade и Hacker Stinger), IS-DOS.
- ZX Format #02 содержит статью о турбо-режимах и их влиянии на совместимость — релевантно для главы про Pentagon vs 48K timing.
- ZX Format #02 также описывает ZX General Sound — музыкальный аддон от X-Trade (!) — может быть релевантно для главы про звук.
- URL: `zxart.ee/eng/software/pressa/newspaper/zx-format/`

### ★★☆ Born Dead (газета, ~2000–2001)
- Сценовая газета с хорошей периодичностью (на начальном этапе).
- Born Dead #0F: правила CC000 compo, результаты CC.999.999, обзор нового трекера.
- Born Dead #10: отчёты с Forever 2e3 (Тренчин, Словакия), Paradox Party, Mekka Symposium 2000. Результаты компо (AY music, intros).
- **Ценность:** Атмосфера вечеринок, результаты компо, контекст сцены рубежа 2000-х. Хорошо для Chapter 15 (The Living Demoscene) и исторического контекста.
- URL: `zxart.ee/eng/software/pressa/newspaper/borndead-0f/` и далее

### ★★☆ Deja Vu (Binary Dimension Association, ~до 2000)
- Один из лучших журналов по оценкам современников. #0A упоминается как «давно не читал таких журналов — огромное количество интересных статей».
- **Что искать:** Систематически просмотреть номера на ZXArt на наличие технических статей.
- URL: искать через `zxart.ee/eng/software/pressa/disk-magazine/` по имени

### ★★☆ Scenergy
- Упоминается в интервью как эталон качества (наряду с Born Dead).
- Команда «серьёзно подошла» к изданию, но проблемы с периодичностью.
- **Что искать:** технические статьи о кодинге, интервью со сценерами.
- Примечание: точные номера на ZXArt нужно искать вручную (поиск по названию).

### ★☆☆ Black Crow #05 (Чёрная Ворона)
- Содержит обзор софта 2000 года, статью о multicolor эффектах на ZX Spectrum (борда + chunks) с примером кода и алгоритмом (!), статью о TR-DOS system variables.
- Multicolor-статья — потенциально релевантна для Part II (эффекты).
- URL: `zxart.ee/eng/software/pressa/disk-magazine/black-crow-05/`

### ★★☆ Subliminal Extacy #01–#05 (Raww Arse → Papaya Dezign, 1995–2023)
- Редкий **англоязычный** сценовый журнал для ZX Spectrum!
- #03 (2001): статьи о Spectrum SE, техники кодинга.
- #05 (2023): самый свежий номер — современная сцена.
- **Ценность:** Англоязычный контент. Если книга будет на английском — прямые цитаты и ссылки проще. Также показывает, что сцена жива и в 2020-х.
- URL: `zxart.ee/eng/software/pressa/disk-magazine/subliminal-extacy-05/`

### ★☆☆ Adventurer
- Несколько номеров на ZXArt (#06, #12 и др.).
- Упоминается как журнал уровнем ниже Scenergy, но с активным сообществом.
- **Что искать:** Возможные технические статьи, обзоры демо.

### ★☆☆ Insanity (Light Force Group)
- Газета, одна из немногих «стоящих» публикаций на рубеже 2000-х (по словам из интервью в Scream/Born Dead).
- Проблема с оперативностью: отчёт с CC появлялся через полгода.

### ★☆☆ Warez #02–#03 (Triumph, 2000–2001)
- Обзоры журналов, софта, игр. Статьи о графических редакторах BGE v2.40 и Hardcore v3.1.
- **Маловероятно релевантно** для книги, но стоит бегло просмотреть.

### ★☆☆ ZX Press #01 (1999)
- «Задача — донести пользователям Speccy как можно больше информации, как для программиста, так и для начинающего.»
- Стоит проверить содержание на ZXArt.

## 3. Другие ресурсы

### bbb.retroscene.org — глобальный архив ZX Spectrum демо
- Файловый архив демо, с возможностью просмотра и скачивания.
- Полезен для получения самих .scl/.tap файлов демо, которые анализируются в книге.

### zxdemo.org (Demotopia) — by Gasman
- Каталог ZX Spectrum демосцены, связан с Demozoo.
- Хороший индекс для поиска конкретных работ, авторов, вечеринок.

### Pouet.net
- Мультиплатформенный каталог демо. Хорош для поиска рейтингов и комментариев к конкретным работам.

### ZX Spectrum Demoscene (книга, Columbia University Press, 2025)
- Marecki, Yerzmyey, Straka. Академическое исследование ZX Spectrum демосцены.
- Этнографический метод, platform studies, media archaeology.
- **Не конкурент** твоей книге (академическая vs практическая), но стоит знать о ней и, возможно, сослаться.
- ISBN: 9788323371250

## 4. Что искать системно на ZXArt

ZXArt хранит полные тексты статей из журналов с переводами. Рекомендуемый подход:

1. **По авторам:** Найти статьи от Dark, Introspec, psndcj, Alone Coder, Shiru, n1k-o, aGGreSSor на ZXArt через поиск по авторам.
2. **По тегам на Hype:** Пройтись по `making of` (самый ценный тег!), `coding`, `z80`, `demo`.
3. **По конкретным журналам:** Для каждого журнала ниже открыть его страницу на ZXArt и просмотреть список статей:
   - Spectrum Expert #01, #02 ✅ (уже есть)
   - ZX Format #01–#08 (особенно #02–#07 — золотой период)
   - Deja Vu (все номера)
   - Scenergy (все номера)
   - Born Dead (все номера)
   - Black Crow (особенно #05)
   - Subliminal Extacy #01–#05
   - Adventurer (все номера)
   - Insanity (все номера)
   - Spectrofon (упоминается в контексте ASC Sound Master)

4. **Поиск по содержимому статей:** ZXArt индексирует содержимое. Можно искать по ключевым словам через сам сайт:
   - `мультиколор` / `multicolor`
   - `ротозум` / `rotozoomer`
   - `3D` / `вращение` / `rotation`
   - `таблица синусов` / `sine table`
   - `размерное кодирование` / `size coding` / `256 байт` / `4K`
   - `AY` / `digital drums` / `beeper`
   - `making of` / `как делалось`
